Reconstrução de uma página de Receitas

Neste desafio você deve reconstruir uma página de receitas qualquer, a receita precisa ter uma parte de ingredientes e outra de modo de preparo, além de uma imagem mostrando como ficaria a receita pronta

Caso sua receita não tenha imagens prontas, busque uma alternativa na internet!

  1. Escolha uma receita da Internet
  2. Crie uma nova página HTML chamado Receita.html.
  3. Comece a recriar a página de receitas usando tags de títulos e listas ordenadas/não ordenadas .
  4. Os ingredientes precisam ser lista não-ordenada.
  5. O modo de preparo precisa ser lista ordenada.
